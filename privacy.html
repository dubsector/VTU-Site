Privacy Policy for VirusTotal Uploader Extension

Last updated: September 30, 2024

Data Collection

The VirusTotal Uploader extension does not collect, store, or transmit any personal information or data from users.

Permissions

The extension requests the following permissions solely to perform its intended functionality:

  • Storage: Used to save user preferences and settings, such as default options for uploads.
  • Tabs: Allows the extension to interact with browser tabs, such as for initiating uploads or scans.
  • Alarms: Used to schedule tasks like checking the status of file uploads.
  • Host Permissions: The extension interacts with the VirusTotal service (https://www.virustotal.com/) to upload files and retrieve scan results.
